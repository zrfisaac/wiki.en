# CPU (Central Processing Unit)

## Description
A **CPU** is the brain of a computer, responsible for executing instructions from software and performing calculations. It processes data, manages input and output operations, and controls system functions.

## [Architecture](#index)

### [aarch64](#index) (ARM64, ARMv8 64-bit)

The 64-bit version of ARM, widely used in smartphones, tablets, embedded systems, and increasingly in servers and desktops (e.g., Apple M1/M2, Raspberry Pi 4).

### [amd64](#index) (x86-64, x64)

A 64-bit extension of x86, introduced by AMD. It allows for larger memory addressing and better performance. Intel later adopted it as Intel64.

### [armv7](#index) (ARM 32-bit, ARMhf)

A 32-bit ARM architecture commonly used in mobile devices, embedded systems, and older Raspberry Pi models. It supports hardware floating-point operations (ARMhf).

### [armv8](#index) (ARMv8-A, 32-bit & 64-bit)

Introduced by ARM, it supports both 32-bit (AArch32) and 64-bit (AArch64) processing. It is the basis for modern ARM chips, including aarch64 (ARM64) used in newer mobile devices, servers, and Apple Silicon (M1/M2).

### [cortex](#index) (ARM Cortex-A, Cortex-M, Cortex-R)

A family of ARM-based CPU cores used in mobile, embedded, and real-time applications. Cortex-A is for high-performance computing, Cortex-M for microcontrollers, and Cortex-R for real-time processing.

### [i386](#index) (x86, 32-bit)

The 32-bit version of the x86 architecture, originally developed by Intel. It supports up to 4GB of RAM and is mainly used in legacy systems.

### [itanium](#index) (x86, 32-bit)

Intel's 64-bit architecture, separate from x86, designed for enterprise and high-performance computing but largely phased out in favor of x86-64.

### [powerpc](#index) (PPC, 32-bit)

A 32-bit RISC architecture developed by IBM, Apple, and Motorola, used in older Macs, embedded systems, and gaming consoles.

### [powerpc64](#index) (PPC64, 64-bit)

The 64-bit extension of PowerPC, used in enterprise servers and high-performance computing.

### [riscv32](#index) (RISC-V 32-bit)

A 32-bit open-source RISC architecture gaining traction in embedded systems and IoT applications.

### [riscv64](#index) (RISC-V 64-bit)

The 64-bit version of RISC-V, designed for high-performance computing, servers, and advanced embedded applications.

### [sparc64](#index) (SPARC 64-bit)

A 64-bit RISC architecture developed by Sun Microsystems, mainly used in enterprise and high-performance computing.
